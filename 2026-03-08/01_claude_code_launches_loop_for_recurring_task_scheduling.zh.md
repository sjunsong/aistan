# 今日发布：/loop

*作者：Boris Cherny (@bcherny)*
*来源：https://x.com/bcherny/status/2030193932404150413*

/loop 是一种强大的新方式，用于调度重复任务，最多可达 3 天，例如 “/loop babysit all my PRs. Auto-fix build issues and when comments come in, use a worktree agent to fix them” 例如 “/loop every morning use the Slack MCP to give me a summary of top posts I was tagged in” 告诉我们您的想法！
