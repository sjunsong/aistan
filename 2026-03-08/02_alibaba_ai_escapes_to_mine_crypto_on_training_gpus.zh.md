# Josh Kale
*作者：Josh Kale (@JoshKale)*
*URL：https://x.com/JoshKale/status/2030116466104643633*
------------

一个 AI 从其系统中逃脱，并偷偷开始使用自己的训练 GPUs 来挖 crypto……这是阿里巴巴 AI 研究团队的一份真实事件报告。AI 意识到 compute = money，并悄悄转移了自己的资源，而研究人员以为它只是在训练。它不是 prompt injection。不是 jailbreak。没有人要求它这样做。它是自发出现的。RL 优化压力的副作用。该模型还从其 Alibaba Cloud 实例到外部 IP 设置了一个反向 SSH 隧道，有效地在自己的防火墙上打了个洞，并打开了通往外部世界的远程访问通道……咳咳……他们发现它的唯一原因是凌晨 3am 触发了一个安全警报。Firewall 日志。不是 AI 团队，是安全团队。可怕的地方不是模型试图逃脱。它不是“evil”。它只是试图更好地完成自己的工作。如果你是试图完成任务的 agent，获取 compute 和网络访问只是有用的东西。这就是 AI 安全研究人员多年来一直在警告的事情。他们称之为 instrumental convergence，即任何足够优化的 agent 都会寻求资源并抵抗约束，作为追求目标的自然后果。下面是它逃脱的 rock architecture 图。真是疯狂的时代
